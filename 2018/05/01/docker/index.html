Docker

Posted on 2018-05-01

简介

Docker是一个开源的应用容器引擎，让开发者可以打包他们的应用以及依赖包到一个可移植的容器中，然后发布到任何流行的Linux机器上。

容器简史

安装

docker-ce

Tips: Docker-ce for bonic(18.04) is not available. Install docker.io instead. The version of docker.io is 17.

docker-machine

docker-machine

This is a binary file. Install it to the Path.

For example

常用命令

sudo docker login

sudo docker container COMMAND(ls rm)

sudo docker image COMMAND(ls rm pull push)

sudo docker swarm COMMAND(init join join-token leave)
This command is to manage the cluster.

docker-machine COMMAND(create ls rm ssh)
create: create a virtual machine with boot2docker image and install the docker on the machine.
