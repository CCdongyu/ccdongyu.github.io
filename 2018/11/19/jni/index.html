Java Native Interface

Posted on 2018-11-19 | Visitors:

写了一个JNI(Java Name Interface)的Demo。

Step

  1. 编写一个带有本地方法的类
  2. 通过javah命令生成jni头文件
  3. 实现jni头文件中的方法，也就是java类中的本地方法
  4. 将c/c++编译为动态连接库
  5. 将生成的动态连接库放置到Java能扫描到的目录
  6. 编译Java类，然后运行
